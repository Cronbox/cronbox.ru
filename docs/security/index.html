Безопасность

С нами безопасно, и вот почему:
  • Сервису не требуется доступ на ваши сервера.
  • Все внешние коммуникации защищены протоколом HTTPS.
  • Сервис работает только на приём данных, ему не требуется внутренний доступ к вашим серверам.
  • В движке сервиса используются актуальные версии современных технологий.
  • Регулярно устанавливаем обновления безопасности для нашей инфраструктуры.
  • Принимаем оплату через известные банковские сервисы, все платёжные данные остаются с вами.
  • Храним только ваш почтовый адрес — для аутентификации — и историю статусов для мониторов.
    Подробно написали об этом в разделе «Персональные данные».
